# How To Use
* You can watch my video here for more details :point_right: [Channel Urproject](https://youtu.be/InADUPpjOok).
* Open your `Command Prompt (CMD)` and `Run as administrator`.
* Type command `slmgr /ipk yourlicensekey` the command to install license, after a popup appears like in the picture.
* Type command `slmgr /skms kms8.msguides.com` that command to connect to kms server, after a popup appears like in the picture.
* Last step type command `slmgr /ato` to activate windows 10 license.
